# kicoach78-crypto.github.io
Application complète de suivi fitness et nutrition. Créez des séances d'entraînement personnalisées, suivez vos exercices en temps réel avec timer intégré, gérez votre apport calorique et vos macros. Historique des performances, supersets, création d'exercices personnalisés, aliments halal, et objectifs personnalisés.
